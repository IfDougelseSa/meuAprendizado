Git intermediario Trabalhando com branches Branche é uma ramificação do projeto. A linha principal é chamada de branch master, ramo principal. Ponto comum onde o branch inicia é chamado de base. git branch Para consultar os branchs existentes no projeto Para criar branchs: git branch "nome_branch" Para mudar de branch git checkout "nome_branch" Para deletar branch git branch -d "nome_branch"
