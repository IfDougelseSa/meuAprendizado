JavaScript em páginas html

Existem duas formas de incluir o JavaScript em Arquivos HTML. Primeira: Diretamente no arquivo HTML5. script codificação javascript script Segunda: A partir de um arquivo externo. script src="meu_script.js" script. Pode criar codigos quantas vezes necessários. A diferenca de criar a tag dentro do head e do body é que as instrucções são carregadas primeiro no head e depois no body alert("mensagem"): Para exibir uma mensagem antes de carregar a página. Precedência de execução: Problemas de precedeência: O codigo ser executado antes do elemento ser criado. Se usar o script no head para determinado elemento html no body, ocorrerá o problema de precedência. Comentários em JavaScript //Comentario de uma linha. /*Comentários de varias linhas */ Variaveis: Espaços em memorias que nos permite armazenar informações temporarias. Variaveis string: Armazenam cadeia de caracteres variaveis number: Armazenam valores numericos Variaveis boolean: Armazenamentos de estados True ou False. Regras para declaração de variaveis em javascript Não podem ser iniciadas por numeros. Não pode ser utilizado caracteres especiais. Não podem ser iguais as palavras das linguagem. var para declarar variaveis. Case sensitive: Diferenciação de palavras maiusculas e minusculas.
